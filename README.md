# AI English Writing Coach

> Agent #106 · [100 AI Agents Challenge](https://hhanng.github.io)

## 🔗 Live Demo
[Open App](https://hhanng.github.io/agent-106-ai-english-writing-coach/)

## ✨ Features
- Essay Coach: word/sentence analysis, overused word detector with upgrade suggestions, writing score, 20 prompts
- Grammar Guide: 30 rules with English + Vietnamese explanations, wrong/correct examples, 30-question quiz
- Vocabulary: 200 words (B1/B2/C1) with Vietnamese definitions, flashcard mode with Easy/Hard tracking, 50 essay phrases
- All progress saved in localStorage

## 🚀 Usage
1. Open the [Live Demo](https://hhanng.github.io/agent-106-ai-english-writing-coach/)
2. Essay Coach: paste your writing → click Analyze → get instant feedback and score
3. Grammar: browse 30 rules with Vietnamese, then test yourself with the 30-question quiz
4. Vocabulary: browse 200 words by level, use flashcards, or copy essay phrases

## 🛠 Tech Stack
- Single-file HTML/CSS/JS — no dependencies, works offline
- localStorage for flashcard progress
